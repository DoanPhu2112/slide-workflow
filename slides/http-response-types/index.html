Success

2xx: request xử lý thành công

Nhóm này không đồng nghĩa mọi thứ giống nhau. Có lúc server trả data ngay, có lúc chỉ xác nhận đã nhận job.

200

OK

Request thành công và trả dữ liệu ngay.

Dùng cho GET, PUT, PATCH phổ biến.
201

Created

Tạo mới resource thành công.

Nên kèm ID hoặc Location header.
202

Accepted

Server nhận request nhưng xử lý async.

Job vào queue, chưa có kết quả cuối.
204

No Content

Request thành công nhưng không cần body.

Hay dùng cho delete hoặc toggle state.

Redirect

3xx: client cần đổi đích hoặc dùng cache

Đây không hẳn là lỗi. Nó báo cho browser, bot hoặc SDK phải tiếp tục sang URL khác hoặc tận dụng bản cache đang còn hiệu lực.

301

Moved Permanently

Resource đã chuyển hẳn sang URL mới.

SEO và bot nên cập nhật link canon.
302

Found

Chuyển hướng tạm thời.

Hay dùng khi điều hướng ngắn hạn.
304

Not Modified

Dữ liệu không đổi, client dùng cache cũ.

Giảm payload và băng thông.
307

Temporary Redirect

Redirect tạm nhưng phải giữ nguyên method.

Quan trọng với POST hoặc PUT.

Client error

4xx: request đến nhưng không hợp lệ

Cùng là lỗi nhưng gốc khác nhau: sai cú pháp, thiếu xác thực, không đủ quyền, đụng validation hoặc vượt quota.

400

Bad Request

JSON sai format hoặc thiếu field bắt buộc.

Request lỗi từ chính phía caller.
401

Unauthorized

Thiếu token hoặc token đã hết hạn.

Chưa qua được lớp xác thực.
403

Forbidden

Đăng nhập rồi nhưng không có quyền.

Auth đúng, permission sai.
404

Not Found

Resource hoặc route không tồn tại.

Không tìm thấy đích yêu cầu.
422

Unprocessable

Cú pháp đúng nhưng data fail validation.

Logic nghiệp vụ reject input.
429

Too Many Requests

Caller đã vượt rate limit.

Server đang tự bảo vệ hệ thống.

Server error

5xx: request đúng nhưng backend hỏng

Đây là lúc nhìn vào application logs, reverse proxy, queue, DB, hoặc timeout giữa các service.

500

Internal Error

Unhandled exception hoặc bug code.

Lỗi phát sinh bên trong app.
501

Not Implemented

Server chưa hỗ trợ capability hoặc method đó.

Feature chưa có hoặc chưa bật.
502

Bad Gateway

Gateway nhận response upstream không hợp lệ.

Proxy chạm upstream lỗi.
503

Unavailable

Service tạm không phục vụ được.

Maintenance hoặc cạn tài nguyên.
504

Gateway Timeout

Upstream xử lý quá lâu nên gateway timeout.

Chain service bị nghẽn hoặc chậm.

Cloudflare edge

52x: nhóm mã phát sinh ở lớp Cloudflare

Chúng giống 5xx ở chỗ request không đi trọn vẹn, nhưng khác ở layer: Cloudflare nằm giữa client và origin server.

520

Unknown Error

Origin trả response bất thường hoặc rỗng.

Cloudflare không parse được phản hồi.
521

Web Server Down

Origin từ chối kết nối từ Cloudflare.

Firewall hoặc app chưa mở cổng.
522

Connection Timed Out

Cloudflare không nhận phản hồi TCP kịp từ origin.

Origin chậm, treo hoặc bị chặn.
524

A Timeout Occurred

Kết nối lên được nhưng origin trả HTTP quá chậm.

Request dài hơn timeout edge cho phép.
