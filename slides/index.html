Slides Workspace

Project decks phải nằm trong subfolder riêng

Không ghi đè trực tiếp vào slides/index.html. File này nên là hub hoặc entry tổng, còn mỗi deck phải nằm ở dạng slides/<project-slug>/index.html cùng CSS đi kèm.

Ví dụ đúng: slides/free-domain-guide/index.html, slides/free-domain-guide/domain-guide.css
Rule thêm: luôn cập nhật hub slides/index.html khi có deck mới.

SSL, Proxy & DNS Foundations

Deck giải thích SSL/TLS handshake, reverse proxy, trust boundaries, OSI layers và DNS resolution cho hệ thống ARP.

free-domain-guide

Đăng ký domain miễn phí

http-response-types

HTTP Response Types

Deck tổng hợp `2xx`, `3xx`, `4xx`, `5xx`, `52x` với flow ví dụ theo phong cách Excalidraw cho từng status tiêu biểu.
